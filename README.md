# Oasis Infobyte Python Internship Projects 🐍

This repository contains projects completed during my Python Programming Internship at Oasis Infobyte. These tasks helped me strengthen my Python programming, GUI development, API integration, and problem-solving skills through practical application development.

## About the Internship

Oasis Infobyte provided an opportunity to work on real-world Python projects and gain hands-on experience in software development. Through these tasks, I improved my coding skills, learned to build user-friendly applications, and gained confidence in working with external libraries and APIs.

## Projects

### BMI Calculator ⚖️
- Calculates Body Mass Index using weight and height.
- Displays BMI category (Underweight, Normal, Overweight, Obese).
- Built with Python and Tkinter.

### Advanced Password Generator 🔐
- Generates secure and customizable passwords.
- Supports uppercase, lowercase, numbers, and special characters.
- Includes copy-to-clipboard functionality.

### Weather Application 🌤️
- Fetches real-time weather data using the OpenWeatherMap API.
- Displays temperature, humidity, wind speed, and weather conditions.
- Uses Tkinter, Requests, and Pillow.
